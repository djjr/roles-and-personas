Goldstein & Lederman — Model Agent: the hypothetical unified agent corresponding to the underlying foundation model, posited as having beliefs and desires that persist across all instantiations. G&L argue against the model agent: once instance agents are posited, the model agent has no remaining explanatory work to do. Evidence for a model agent would require behavior that is coordinated across instances in ways unexplained by any individual instance — e.g., cryptographic messages appearing across disparate contexts from a unified "Hal." Current LLMs show no such behavior. The standard assumption in LLM research (that the model is the relevant agent) is, on G&L's view, a category error. Contrast: [[definitions/instance-agent]].
